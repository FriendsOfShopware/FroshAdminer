Mit den Adminer Plugin kannst du ganz einfach auf die Datenbank oder auf dein Elasticsearch Server verbinden. Nach der Installation erhälst du einen neuen Menüpunkt unter Einstellungen. Wo du dann auch automatisch eingeloggt wirst.

Das Plugin wird von der Github Organization FriendsOfShopware entwickelt. Maintainer des Plugins ist: Soner Sayakci
